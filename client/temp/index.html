  Speed

We chose node.js for its incredible speed based on the Google V8 Javascript engine. You simply will not find this type of speed with PHP.

  Versatility

Yeti is open-source and completely transparent. If you know Javascript and node.js, feel free to modify it for your own needs.

  Security

Yeti uses a proven and very secure user system based on bcrypt, a key derivation function that was introduced in 1999 and has since been un-crackable. Your security is important, why sacrifice?

  Simplicity

While Yeti does not have thousands of plugins, it does have simplicity. The core code is very simple and follows a Restful API pattern that is extremely easy to understand and very scalable.

  Mongo

Yeti uses the extremely fast NoSQL database MongoDB. This document-based database offers unparalleled speed, built in clustering, and the flexibility to host all of your content anywhere in the world.

  AngularJS

Yeti uses Angular for routing and it's very robust structure. This allows for things like partial views and two-way data-binding for extremely powerful data processing and displaying.

  Express

Yeti uses ExpressJS for server-side routing and API end-points. Express is proven, stable, fast, and easy to use.

  Custom

Edit CSS and Javascript right inside of the full-featured administration panel. No need to wade through and keep track of all of these files. You can even edit them on the go from your mobile device!
